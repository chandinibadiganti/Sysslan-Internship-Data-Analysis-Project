# Sysslan Internship - Data Analysis Project 🚆

## Internship Project
This project was completed as part of my Data Analytics internship at Sysslan.

## Project Overview
The project focuses on analyzing train journey data to identify patterns in journey duration, distance relationships, route performance, and station traffic using Python-based data analysis and visualization techniques.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Data Visualization

## Tasks Completed

### 1. Actual vs Predicted Journey Duration
Compared actual journey duration with predicted values to analyze model performance.

### 2. Distance vs Journey Duration Analysis
Studied the relationship between travel distance and journey duration.

### 3. Route Performance Analysis
Analyzed average journey duration across different starting stations/routes.

### 4. Station Traffic Analysis
Identified stations with the highest train traffic based on arrivals and departures.

## Project Files

- `Sysslan_Data_Analysis_Project.ipynb` - Complete analysis notebook
- `Dataset1.csv` - Dataset used for analysis
